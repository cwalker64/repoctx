# Examples

Small, runnable scripts. Each indexes a real corpus and prints results — no
setup, no network, no API keys.

| Script | What it shows |
| --- | --- |
| [`quickstart.py`](quickstart.py) | Index a directory and run semantic searches. |
| [`build_graph.py`](build_graph.py) | Build the dependency graph and walk a symbol's neighbours. |

Run any of them from the project root, e.g.:

```bash
python examples/quickstart.py
```
